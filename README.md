# Automacao-de-Testes-com-Cypress
Projeto prático para treinar automação de testes inicial com Cypress, onde foi utilizado o site https://devfinance-agilizei.netlify.app/# para testes de transações financeiras (entradas e saídas).

Foi usado como base os ensinamentos obtidos através do canal do YouTube https://www.youtube.com/@Agilizei , na playlist Cypress direto ao ponto.
